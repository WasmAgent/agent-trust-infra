# Contributing

Thank you for your interest in Agent Trust Infrastructure. This document
provides guidelines for contributing to this repository.

## Getting started

1. Clone the repository.
2. Run `bun install` to install dependencies.
3. Run `bun test` to verify your setup.

## Patrol tags

Issues filed by the patrol daemon include `<!-- patrol: ... -->` and
`<!-- confidence: ... -->` tags. For the standard operating procedure on how to
interpret and respond to these tags, see
[docs/OPERATIONS.md](docs/OPERATIONS.md).

## How to contribute

- **Bug reports:** Use the [bug report template](.github/ISSUE_TEMPLATE/bug_report.yml).
- **Spec proposals:** Use the [spec proposal template](.github/ISSUE_TEMPLATE/spec_proposal.yml).
- **Pull requests:** Open a PR with a clear description of the change and
  reference any related issues.

## Code of conduct

Be respectful, constructive, and inclusive. Harassment or discriminatory
language will not be tolerated.
